Modules, Classes, Mixins and Enums

RaptorJS provides a simple library to support defining namespaced modules, classes, mixins and enums. This library uses a clean and easy-to-understand syntax based on the Asynchronous Module Definition (AMD) syntax. RaptorJS provides an AMD implementation that extends AMD in backwards compatible ways to provide new features that make building modular JavaScript applications even easier. The syntax is is very close to pure JavaScript, but fills gaps in the existing JavaScript language. Now you can start creating modular code that is easy to maintain without waiting for a new version of JavaScript or switching to an alternate language.

Features

  • AMD-compliant
  • Makes JavaScript Better
  • Clean Syntax
  • Lightweight AMD Implementation
  • Extends AMD with the following:
    • Classes and Class Inheritance
    • Enumerations
    • Extensions
  • Code Reuse across Client and Server
  • Lazy Initialization
  • Supports Parallel Downloading
  • Cross-platform
  • Decoupled from Async Module Loader
