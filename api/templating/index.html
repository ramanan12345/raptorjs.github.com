templatingDefined in: /templating/templating.js

This module provides the runtime for rendering compiled templates.

The code for the Raptor Templates compiler is kept separately in the templating.compiler module.

render( templateName, data, context )

Renders a template to the provided context.

The template specified by the templateName parameter must already have been loaded. The data object is passed to the compiled rendering function of the template. All output is written to the provided context using the "writer" associated with the context.

Parameters

  • templateName- The name of the template to render. The template must have been previously rendered
  • data- The data object to pass to the template rendering function
  • context- The context to use for all rendered output (required)

renderToString( String templateName, Object data, templating$Context context )

Renders a template and captures the output as a String

Parameters

  • templateName: String - The name of the template to render. NOTE: The template must have already been loaded.
  • data: Object - The data object to provide to the template rendering function
  • context: templating$Context - The context object to use (optional). If a context is provided then the writer will be temporarily swapped with a StringBuilder to capture the output of rendering. If a context is not provided then one will be created using the "createContext" method.
