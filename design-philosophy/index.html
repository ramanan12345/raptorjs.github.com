    Design Philosophy

    Overview

    As an eBay project, RaptorJS has been designed to perform extremely well by keeping the code small and efficient, as well as supporting the industries best practices. In addition, we believe that developers should have the freedom to use any JavaScript library and framework that they want so RaptorJS was designed to work alongside other technologies to simplify development further.
