Overview

RaptorJS is a modular JavaScript toolset that provides a foundation for building any type of web application and was designed to work with your existing JavaScript libraries—not replace them. Since it is a toolset and not a framework, feel free to take what you want and save the rest for later.

A feature that is unique to RaptorJS is adapative packaging. This feature allows developers to easily target specific code for different browsers, devices, frameworks, JavaScript runtime environments, etc. By simply separating out environment-specific code into individual files and creating RaptorJS package files that describe dependencies (including which dependencies are specific to a particular environment), developers can create JavaScript modules and UI components that function correctly in multiple environments. Adaptive packaging has allowed the RaptorJS toolset to be portable across multiple platforms (including Node, Rhino and all web browsers).

RaptorJS ships with a number of helpful modules, including Raptor Templates and the Widgets Framework, and we hope that you, too, will take advantage of RaptorJS to build and share JavaScript modules and UI components with others. RaptorJS is being used on the most heavily used eBay pages and will continue to be improved to support world-class websites and make web application development easier for all developers. The RaptorJS toolset is still in development, and we welcome feedback and contributions.
